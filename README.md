# orange-parent

### 组织结构

```
orange-parent
├── orange-common -- 公共模块
├── orange-config-repo -- 统一配置中心
└── orange-microservice -- 集中管理微服务
	  ├── orange-config-server -- 配置中心服务端
	  ├── orange-eureka -- 服务注册中心
	  ├── orange-sso -- 单点登录
	  ├── orange-user -- 用户数据模块
	  └── orange-user-web -- 用户个人中心模块
```
